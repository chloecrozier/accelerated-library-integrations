# Results

Save benchmark CSVs here after running on each target platform.

Recommended filenames:

- `brev_l4.csv`
- `dgx_spark.csv`
- `nsys_capture.csv`

The CSV columns are:

- `platform_label`
- `benchmark`
- `mode`
- `device_name`
- `cudnn_version`
- `batch_size`
- `latency_ms`
- `throughput_img_s`
- `speedup_vs_cpu`
